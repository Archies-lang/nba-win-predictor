# Commands to push to GitHub

# After creating your repository on GitHub, run these commands:

# 1. Add remote origin (replace 'Archies' with your GitHub username)
git remote add origin https://github.com/Archies/nba-win-predictor.git

# 2. Push code to GitHub
git push -u origin main

# 3. Verify the push worked
git remote -v

# Alternative: If you have a different repository name, use:
# git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
